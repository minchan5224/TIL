> #### 4. Python 기초
>> #### 리스트와 딕셔너리 복사..
>>> ```Python
>>> d1 = {'k':1}
>>> a1 = d1
>>> d1['k'] = 2
>>> print(a1)
>>> """
>>> {'k': 2}
>>> """
>>> ```
>>> 값을 복사하는 것이 아닌 같은 메모리 주소를 가리킨다. 
>>> - 따라서 copy.deepcopy()를 이용하자. list도 마찬가지다.
>>> 
>>> ```Python
>>> import copy
>>> d2 = {'k':1}
>>> a2 = copy.deepcopy(d2)
>>> d2['k'] = 2
>>> print(a2)
>>> """
>>> {'k': 1}
>>> """
>>> ```
>>> 정상적으로 동작한다.
