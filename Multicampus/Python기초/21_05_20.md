>> ## HTML
>> - 문서의 기본구조
>>> ```HTML
>>> <!DOCTYPE html>
>>> <html>
>>> <head>
>>>     <title>타이틀</title>
>>> </head>
>>> <body>
>>>     <h1>저의 첫 웹사이트에 오신것을 환영합니다.</h1>
>>> </body>
>>> </html>
>>> ```
>> - 태그의 구조
>>> <태그>내용</태그>
>>> 
>>> <태그 속성-"속성값">내용</태그>
>>> 
>> - 하이퍼텍스트
>>> ```HTML
>>> <a href="next.html">다음페이지</a>
>>> <a href="http://naver.com" target="_blank">다음페이지</a>
>>> ```
>>> href : 이동할 주소나 페이지
>>> 
>>> target : 페이지를 어떻게 열것인가.
>>> - ```_blank ``` : 새창
>>> 
>>> - ```_self ``` : 현재페이지
>>> 
>>> - ```_parent ``` : 부모프레임
>>> 
>>> - ```_top ``` : 가장 상위프레임
>>>
>> - 목록태그
>>> ```HTML
